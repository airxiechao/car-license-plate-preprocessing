### Car License Plate Recognition


#### Process

1. ROI Selection
    - Automatic plate detection:
        - Using ALPR
        - Using HAAR cascade classifier
    - Manual
    - Semiautomatic: https://youtu.be/BXO8fdkvCWM
2. Plate segmentation and affine transformation
3. Noise removal
4. Binarization 
5. Morphology operations
6. Segmentation
7. Enhancement & Final
